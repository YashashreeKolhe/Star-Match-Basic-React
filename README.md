# Star-Match-Basic-React
Star-Match is a simple game developed to get basic of React concepts like function/class components, states, props, custom hooks, etc.

## Pre-requisites

* Node.js installation: v12.16.3  https://nodejs.org/en/download/

## How to run the application

Clone the repository, open the terminal and navigate into the directory where you have cloned the repo.
Run the following commands,
```
npm install
npm run start
```
You will have the UI running on https://localhost:3000
